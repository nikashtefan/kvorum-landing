# Ресёрч: потенциальные сегменты для Кворум

**Дата:** 2026-02-26
**Для:** Глеб Хардеев
**Контекст:** Кому ещё продавать интенсив по вайбкодингу кроме текущей ЦА

---

## 1. Продакт-менеджеры (PM)

### Почему им это нужно
- **Главный пейн**: зависимость от разработчиков. PM придумал гипотезу — надо ждать спринт, чтобы проверить
- Вайбкодинг позволяет PM самому собрать прототип/MVP и проверить гипотезу за день
- ACM (Communications of the ACM) уже назвали это "Vibe Coding Imperative for Product Managers"
- Jackie Bavaro (автор "Cracking the PM Interview") ведёт серию "Vibe Coding for Product Managers"

### Конкретные кейсы для PM
- Лендинг для проверки гипотезы (вместо Tilda)
- Дашборд с метриками продукта
- Прототип фичи для юзер-тестирования
- Бот для сбора обратной связи от пользователей
- Калькулятор unit-экономики

### Где их искать
- **Telegram**: ProductStar (@bootcamp_ps), ProductCamp Russia, Product Gym Ru (@productgym), канал Ани Булдаковой, Produktkhaki (@producthacking от ФРИИ)
- **Оффлайн**: ProductCamp (крупнейшее сообщество PM в РФ и СНГ)
- **Курсы**: ВШЭ "Продакт-менеджер", Нетология, ProductStar
- **vc.ru**: раздел "Продукт"

### Оценка сегмента
- **Платёжеспособность**: высокая (зарплаты PM 150-300K+)
- **Мотивация**: очень высокая — прямо решает их проблему
- **Размер**: большой (тысячи PM только в Москве)
- **Конкуренция**: средняя — уже есть курсы, но мало кто делает практический интенсив с менторством
- **Вердикт: горячий сегмент**

---

## 2. Предприниматели / фаундеры

### Почему им это нужно
- Главный пейн: нет денег на разработчика (или разработчик дорогой и медленный)
- MVP за выходные вместо месяцев — проверить идею до вложения бюджета
- Biz360 уже пишет про вайбкодинг как инструмент для бизнес-задач

### Конкретные кейсы для предпринимателей
- MVP приложения/сервиса для проверки идеи
- Лендинг + бот для лидогенерации
- CRM-лайт или система учёта под себя
- Автоматизация рутины (сбор заказов, отчёты, email-рассылки)
- Калькулятор стоимости для клиентов

### Где их искать
- **Telegram**: каналы для малого бизнеса (подборка 119 каналов на vc.ru), Т-Ж бизнес, МТС Медиа для предпринимателей
- **Оффлайн**: бизнес-завтраки, акселераторы (ФРИИ, Сколково)
- **vc.ru**: основная тусовка стартаперов
- **Нетворкинг**: бизнес-клубы, коворкинги

### Оценка сегмента
- **Платёжеспособность**: разная (от нуля до высокой)
- **Мотивация**: очень высокая — экономия сотен тысяч на разработке
- **Размер**: огромный
- **Конкуренция**: высокая — много курсов "запусти бизнес с AI"
- **Вердикт: перспективный, но надо отстроиться от инфобиза**

---

## 3. Маркетологи

### Почему им это нужно
- Главный пейн: "поставь задачу разработчику и жди" — на любую мелочь на сайте
- Skillbox уже выпустил материал "Вайбкодинг для маркетологов"
- Могут за день реализовать идею: калькулятор, виджет, интерактивный блок, бот

### Конкретные кейсы для маркетологов
- Telegram-бот для лидогенерации / прогрева
- Лендинг под акцию за час (без Tilda-ограничений)
- Калькулятор стоимости на сайт
- Дашборд аналитики по соцсетям
- Парсер/скрапер для мониторинга конкурентов
- Email-автоматизация

### Где их искать
- **Telegram**: каналы по маркетингу и digital
- **Конференции**: РИФ, Стачка, Digital October
- **Курсы**: Skillbox, Нетология (пересечение аудитории)
- **vc.ru**: раздел "Маркетинг"

### Оценка сегмента
- **Платёжеспособность**: средняя (зарплаты 80-200K)
- **Мотивация**: высокая — снимает зависимость от dev-команды
- **Размер**: очень большой
- **Конкуренция**: средняя
- **Вердикт: хороший сегмент, но нужны маркетинговые кейсы**

## 4. Аналитики данных

### Почему им это нужно
- Уже работают с данными, но ограничены Excel/BI-инструментами
- Вайбкодинг = возможность делать кастомные инструменты: дашборды, отчёты, автоматизации
- Кейс safe-deal из лендинга — прямо их язык

### Конкретные кейсы для аналитиков
- Автоматические отчёты (вместо ручного Excel)
- Кастомный дашборд с визуализациями
- Скрипт для ETL / обработки данных
- A/B-калькулятор
- Инструмент для когортного анализа

### Где их искать
- **Telegram**: каналы для аналитиков (подборка на Нетологии)
- **Хабр**: раздел "Аналитика"
- **Курсы**: Яндекс.Практикум, Нетология (пересечение)

### Оценка сегмента
- **Платёжеспособность**: высокая (зарплаты 150-250K+)
- **Мотивация**: средне-высокая (уже имеют часть навыков)
- **Размер**: средний
- **Вердикт: нишевый, но платёжеспособный**

## 5. HR / рекрутеры

### Почему им это нужно
- Пример из Replit: HR-менеджер Kelsey за 3 дня собрала свой org chart tool, потому что все существующие решения не подходили
- Могут автоматизировать рутину: скрининг резюме, трекинг кандидатов, онбординг

### Конкретные кейсы для HR
- Бот для первичного скрининга кандидатов
- Дашборд по воронке найма
- Система онбординга нового сотрудника
- Инструмент для ревью job descriptions
- Self-service инструмент для менеджеров (отпуска, справки)

### Где их искать
- **Telegram**: HR-каналы, рекрутинговые сообщества
- **Конференции**: HR Tech, People Analytics

### Оценка сегмента
- **Платёжеспособность**: средняя
- **Мотивация**: средняя (меньше tech-awareness)
- **Размер**: средний
- **Вердикт: потенциальный, но нужна сильная адаптация месседжа**

## Рекомендация

**Топ-1 приоритет — продакт-менеджеры:**
- Самая горячая боль (зависимость от разработки)
- Высокая платёжеспособность
- Ника сама Lead PM — идеальный "свой человек" для этой аудитории
- Есть готовые каналы дистрибуции (ProductCamp, TG-сообщества PM)
- Кейсы на лендинге уже резонируют (A/B кейс-стади, дашборды)

**Топ-2 — предприниматели**, но нужно чётко отстроиться от инфобиз-курсов. Акцент на "инструмент, не мечта".

**Для маркетологов** стоит сделать отдельный лендинг/посыл с их кейсами (бот, лендинг за час, калькулятор).
